Cap Defend V17d

문서 갱신: 2026-03-25 적용 버전: V17d 시간 기준: KST (미국장은 현지 정규장)

1. 핵심 규칙

  1. 자산 배분: 주식 60 : 코인 40 (실투입 58.8 : 39.2 : 현금 2%)
  2. 시그널 t-1 종가, 체결 t일 — Look-Ahead Bias 금지
  3. 매도 먼저, 매수 나중
  4. Delta 매매: 목표와 현재의 차이만큼만 거래
  5. 트리거(Crash/플립)는 즉시, 종목 교체는 앵커일에만
  6. rebalancing_needed 플래그: 목표 달성까지 재시도, 달성 후 방치
  7. 현금 2% 마진: 주식/코인 모두
현재 운영 상태: 코인 자동매매 운영 중 / 주식 자동매매 운영 중 (소액) / 텔레그램 알림 활성

3. 60:40 자산 배분

위험자산 목표: 주식 60 / 코인 40
실투입: 주식 58.8% / 코인 39.2% / 현금 2% (자연 잔여)

±5%p 밴드: 주식 비중이 53.8~63.8% 밖이면 계좌 간 자금 이동.
기준: 월 1회 이상 확인. 자산관리 페이지에서 조회 버튼으로 자동 계산.

4.2 텔레그램 경보 수신 시

에러 알림: 로그 확인 후 수동 개입 여부 판단.

5. 자동매매 시스템

5.1 실행 스케줄 (KST)

시각 코인 주식
09:15 recommend_personal.py: 신호 계산 + HTML + signal_state.json + 텔레그램
09:20 --trade (매매) -
매 :05/:35 --monitor (긴급+복구) -
23:35 - --trade (4트랜치 매매)
00:05~06:05 - --monitor (리밸런싱 복구, 매시)

모두 run_trade.sh flock 래퍼로 동시실행 방지.

5.2 주문 실행 패턴 (공통)

원칙: 매도 우선, 목표 delta 기준, 주문 후 잔고 재조회
for attempt in 1..5:
  1. 현재가 조회
  2. 주문 (주식: 지정가 ±1%, 코인: 시장가 분할)
  3. 5초 대기
  4. 미체결 전량 취소
  5. 잔고 조회 → 체결량 확인 (before ± delta 추적)
  6. 목표 달성? → 종료
  7. 미달 → 다음 시도

5.3 --trade vs --monitor

--trade (1일 1회) --monitor (수시)
앵커 체크 앵커일이면 트랜치 목표 갱신 → rebalancing_needed=true 안 함
트리거 (Crash/플립) 체크 + 즉시 처리 코인: Crash/카나리/DD 체크. 주식: 미구현 (09:15 종가 기준)
매매 (rebal=true) delta 매매 미체결 취소 → 재계산 → 새 주문
매매 (rebal=false) 앵커 날짜만 확인 미체결 확인만

5.4 리밸런싱 플래그

주식(kis_trade_state.json)과 코인(trade_state.json)은 별도 상태 파일에서 독립 관리.

true가 되는 상황: 앵커일 도달, 카나리 플립, Crash 복귀, DD Exit, 매도 실패(잔량)

false가 되는 상황: 목표 달성 (금액 ±5% or 주수 일치), Crash 전량 매도 성공, pending 모두 완료

false이면: 다음 앵커까지 가격 변동 무시. 매매 안 함.

8. 용어집

용어 설명
카나리 시장 추세를 판단하는 선행 지표. 위에 있으면 공격(Risk-On), 아래면 방어(Risk-Off)
Hysteresis (히스테리시스) 노이즈 필터. 전환 시 ±N% 밴드를 넘어야 상태가 바뀜
EW (동일비중) 선정된 종목을 같은 비중으로 보유
Z-score 모멘텀과 Sharpe를 표준화하여 합산한 점수. 높을수록 우선
Crash Breaker (급락 차단) 기준 지수가 급락하면 전량 매도하고 일정 기간 현금 대기
DD Exit (낙폭 이탈) 보유 종목이 최근 고점 대비 크게 하락하면 매도
Blacklist (급락 제외) 하루에 크게 빠진 종목을 일정 기간 매수 대상에서 제외
PFD (플립 후 재평가) 카나리 전환 후 N일 뒤 종목을 다시 확인하여 리밸런싱
트랜치 자산을 N등분하여 서로 다른 날에 리밸런싱. 타이밍 리스크 분산
앵커일 각 트랜치가 리밸런싱하는 날짜 (예: 1일, 8일, 15일, 22일)
rebalancing_needed 목표 포트폴리오와 현재가 다를 때 true. 달성하면 false
Delta 매매 전량 매도→재매수가 아닌, 목표와 현재의 차이만큼만 거래
TX 백테스트에서 가정한 편도 수수료+슬리피지 (주식 0.2%, 코인 0.4%)
